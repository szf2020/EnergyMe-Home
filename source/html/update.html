Firmware Rollback

You can revert to the previous firmware version stored in the other partition. This is useful if the current firmware has issues.

⚠️ Warning: This will restart the device and switch to the previous firmware version.

Backup Configuration

Download the current device configuration as a JSON file. This includes all NVS settings (channels, MQTT, InfluxDB, etc.)

Backup Filesystem

Download all LittleFS files (energy data, logs, etc.) as a TAR archive. No temporary files are created on the device.

Restore Configuration

Upload a configuration backup JSON file to restore device settings. This will restore all NVS configuration data.

⚠️ Warning: Device will restart after upload completes. Configuration is applied during boot.

Restore Filesystem

Upload a LittleFS backup TAR file to restore all files (energy data, logs, etc.). Existing files will be overwritten.

⚠️ Warning: This will overwrite existing files with the same names from the backup.
